# Website Template ballstt

A clean static website template repository for portfolio and product landing page demos.

## Included Template

`template1` is an Apple-inspired one-page portfolio/product layout built with plain HTML, CSS, and JavaScript.

## Quick Start

1. Open `template1/index.html` in your browser.
2. Replace placeholder product copy with your own content.
3. Update links, pricing blocks, and section labels to match your project.

## Project Structure

- `template1/index.html`: Main landing page template.
- `template1/DESIGN.md`: Design language reference used to build the page.
- `template1/preview.html`: Light-mode token preview from source design pack.
- `template1/preview-dark.html`: Dark-mode token preview from source design pack.
- `template1/README.md`: Template-specific notes .
