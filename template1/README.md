# Template 1: Portfolio Product Landing Page

This folder contains a polished single-page portfolio/product website template based on an Apple-inspired visual language.

## Files

- `index.html`: Main website template ready to customize.
- `DESIGN.md`: Style reference (typography, colors, spacing, components).
- `preview.html`: Light-mode design token reference page.
- `preview-dark.html`: Dark-mode design token reference page.

## Customize

1. Edit headline, section copy, and CTA labels in `index.html`.
2. Replace placeholder specs, pricing, and case study text.
3. Add your own assets/screenshots as needed.

## Notes

- Content is placeholder/demo content by default.
- The page is fully static and does not require a build step.
